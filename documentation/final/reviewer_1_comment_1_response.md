# Reviewer 1, Comment 1 Response Draft

We thank the reviewer for requesting clearer description of the study design, replication, and statistical analysis across the bioinformatic, in vitro, and in silico components of the study. We revised the manuscript to add a dedicated experimental-design and replication subsection and to state explicitly that the mRNA-seq, miRNA-seq, and circRNA microarray assays each used three independently cultured and treated biological replicates per condition. We define the independently cultured and treated sample as the statistical unit and clarify that technical measurements, where applicable, were averaged within biological replicate rather than treated as independent observations.

We also expanded the computational Methods to describe assay-specific normalization, filtering, and statistical testing. The revised mRNA-seq analysis uses edgeR with TMM normalization, CPM filtering, negative-binomial generalized linear models, and likelihood-ratio testing. The revised miRNA-seq analysis uses log2(CPM + 1) values and predefined two-sided Welch tests. The revised circRNA analysis uses the recovered original circRNA differential-expression tables as the Figure 2 source and Arraystar normalized log2 expression values for heatmap source data. For each assay, nominal P values and Benjamini-Hochberg FDR values are retained and exported.

The revised manuscript distinguishes exploratory nominal discovery from FDR significance. Nominal-discovery features are defined by nominal P < 0.05 and absolute fold change >= 1.5, while FDR-significant features are defined by BH FDR < 0.05 and absolute fold change >= 1.5. We clarified that nominal-discovery features were used for broad candidate generation and exploratory network construction, and that they should not be interpreted as genome-wide or FDR-significant discoveries unless they also satisfy the FDR criterion.

We further clarified the sequence of computational analyses: differential expression, interaction prediction with miRanda, exploratory network construction, biological prioritization of fibrosis-relevant candidates, and experimental evaluation of selected predictions. The revised text does not claim that every predicted interaction was experimentally validated; instead, it states that selected predictions were experimentally evaluated and that these experiments produced both supportive and negative or inconclusive results.

To make these revisions transparent, we added brief replicate statements to the Results, expanded computational figure legends, prepared a supplementary experimental-design table, and organized processed data, code, analysis parameters, enrichment inputs, miRanda inputs and parsed outputs, network node and edge tables, figure source-data files, and rerun instructions in a reproducible repository.
