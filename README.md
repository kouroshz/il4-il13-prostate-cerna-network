# IL-4/IL-13 ceRNA Network Reproducible Analysis

This project organizes the computational code, compact processed inputs, final outputs, computational provenance, and manuscript-support materials for the IL-4/IL-13 prostate stromal fibroblast ceRNA manuscript revision.

The final public workflow recomputes differential expression from compact processed matrices:

- `data/processed/mrna/all_counts.xlsx`
- `data/processed/mirna/Expression_Browser_CPM.csv`
- `data/processed/circrna/circRNA_original_IL4_vs_Vehicle_DEG.tsv`
- `data/processed/circrna/circRNA_original_IL13_vs_Vehicle_DEG.tsv`
- `data/processed/circrna/CircRNA_normalized_Data_KZ.xlsx`

The revised manuscript workflow uses recovered original analyses where valid: edgeR for Figure 3 mRNA values, the verified CPM/Welch-test workflow for miRNA source data, and recovered original circRNA DEG tables for Figure 2 values. FDR values are exported for transparency, while exploratory candidate generation uses the nominal-P framework described in `config/analysis.yml`.

## Reproduce Final Workflow

From the project root:

```sh
Rscript run_all.R
```

or:

```sh
make run
```

The workflow generates final differential-expression tables, gprofiler2 outputs, miRanda provenance tables, the curated integrated ceRNA network source table, Cytoscape-compatible source tables, figures, source data, and `sessionInfo.txt`. The cleaned `github_release/` copy omits bulky duplicate artifacts such as generated PDFs, cached `.rds` responses, and full internal validation tables; reproducible outputs are regenerated by `Rscript run_all.R` where applicable.

## Key Directories

- `scripts/`: final executable workflow scripts.
- `R/`: shared helper functions.
- `config/analysis.yml`: thresholds, inputs, and analysis settings.
- `data/metadata/`: explicit assay sample metadata.
- `data/processed/`: compact inputs, recovered miRanda resources, and final generated result tables.
- `results/final/`: generated figures, enrichment outputs, network validation plots, and source data.
- `documentation/final/`: manuscript-ready methods/results text, reviewer-response notes, validation reports, and decision packages.
- `github_release/`: cleaned public release copy.

## Network And miRanda Scope

The repository now includes a curated integrated IL-4/IL-13 ceRNA network source table parsed from Jill/Quentin's integrated network workbook: `data/processed/final/integrated_ceRNA_network_curated.tsv`. This table is treated as the primary source for the final pro-fibrotic shared Cytoscape network, especially Figure 4D, and includes miRanda scores and predicted binding energies for candidate interactions. Original coordinate-like workbook columns are preserved, but their interpretation is not claimed without Quentin confirmation.

The Figure 4D network is a biologically filtered pro-fibrotic subnetwork centered on CEMIP, TNC, and LIFR. It is exploratory and hypothesis-generating. Edges represent miRanda-predicted interactions, not experimentally proven physical competition for every edge. The raw IL-4 and IL-13 miRanda DOCX files are retained as incomplete supporting provenance rather than the only authoritative network source.

Mature miRNA FASTA inputs must use RNA bases (`U`), not DNA `T`, for miRanda binding-energy prediction. The canonical mature human miRNA FASTA in the public release is `data/processed/miranda_inputs/mature_hsa.fa`; duplicate mature FASTA copies are intentionally not kept.

The curated integrated network table provides the miRanda score and energy values used for the prioritized miR-140-3p/CEMIP/circPAPPA, miR-135b-5p/TNC/circLIFR, and miR-625-3p/LIFR axes. A de novo transcriptome-wide miRanda search is not required to reproduce the revised manuscript source data in this release.

## Candidate Highlighting

miRNA volcano plots use nominal P < 0.05 for the main exploratory coloring. miR-140-3p, miR-135b-5p, and miR-625-3p are highlighted with an explicit biologically prioritized/tested category in the source data and figures; this is separate from the nominal-P rule and is not a hidden threshold relaxation.
